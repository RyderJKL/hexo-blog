0x00 select的使用

语法：

{ Select [Top(数值)] 字段列表 *From 表 [Where 条件 ][Order By 字段] [Group 字段] }

其中:

  • order by:按字段排序
  • group by 按字段求和
选取全部数据
选取指定字段的数据
只选取前3条数据
用原表中的字段产生派生字段
根据条件选取
关键字查询
查询结果排序

0x02 update的使用

语法：

{ Update 表 Set 字段1=字段1值，字段2=字段2值，……[Where 条件] }

0x03 Delete的使用

语法：

{ Delete From 表 [Where 条件] }

删除一个表
按条件删除

0x04 like模糊查询

“-”“[]”(特别注意只有 char varchar text 类型的数据才能使用like运算符和通配符）

like相当于=

select from table where name not like ‘计算机’

select name,scout,ctest
from course
where chame like ‘计算机%’ //查询以course表中差么字段以计算机开头的所有开头匹配查询
like ‘%计算机’ //结尾匹配查询
like ‘%中间匹配%’ //中间匹配查询
like ‘计算机%基础’ //使用通配符%查询以“计算机”开头“基础”结束的所有信息

select cname，scount,ctest
from couse
where cname like ‘计算机—‘
and not cname like ‘计算机–’

select cname,scount,ctest
from course
where cname like ‘[^计生]%’
order by cname desc

like ‘%m%’ escape ‘m’
`

用escape定义了转义字符 m 所以第二个%被作为”百分号“而不再作为sql的关键字。
