### Object
其实很多类型都是 Object 类型的实例，虽然 Object 类型的实例不具备多少功能，但是对于应用程序中存储和传输数据而言，确是非常理想的选择。

有两种方法创建 Object 实例:一个是使用 `new Object`，即是调用 Object 的构造函数。而另一种方式是使用对象字面量。

如果使用字面量的方法，则不会调用 Object 的构造函数。
